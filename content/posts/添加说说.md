## 兼容 Cloudflare

- 新增说说功能已经添加到 shuoshuo.astro
- 后台管理入口已接入 shuoshuo.ts
- 公开页面访问：`/shuoshuo`
- 后台管理访问：`/api/admin/shuoshuo`

## 部署到 Cloudflare

如果你已经在 Cloudflare 托管，建议执行：

1. `npm run db:migrate:remote`
2. `npm run deploy`

> 这会把新增的 `shuoshuo_posts` 数据表同步到远程 D1，并部署最新代码。
